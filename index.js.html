JavaScript-Errores

Tipos de Errores JS

Al ejecutar código JavaScript, pueden ocurrir diferentes errores.
Los errores pueden ser errores de codificación cometidos por el programador, errores debido a una entrada incorrecta y otras cosas imprevisibles.

Try declaración define un bloque de código para ejecutar (para probar).
Catch declaración define un bloque de código para manejar cualquier error.
Finally declaración define un bloque de código para ejecutar independientemente del resultado.
Throwinstrucción define un error personalizado.
